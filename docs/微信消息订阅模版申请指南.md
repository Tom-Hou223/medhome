# 药效记 - 微信消息订阅模版申请操作指南

## 一、前置准备

### 1.1 必备条件
- 已注册微信小程序账号
- 小程序已完成基础信息配置
- 具有小程序管理员权限

### 1.2 所需信息
- 小程序 AppID
- 小程序 AppSecret

---

## 二、申请订阅消息模版

### 2.1 登录微信公众平台

1. 访问 [https://mp.weixin.qq.com](https://mp.weixin.qq.com)
2. 使用管理员微信扫码登录
3. 选择对应的小程序

### 2.2 进入订阅消息管理

1. 在左侧菜单栏找到 **功能** → **订阅消息**
2. 点击进入订阅消息管理页面

### 2.3 选择公共模版库

点击页面右上角的 **公共模版库** 按钮

---

## 三、申请用药提醒模版

### 3.1 搜索模版

在搜索框输入关键词：**用药提醒** 或 **服药提醒**

### 3.2 选择合适的模版

推荐选择包含以下字段的模版：

**模版示例：**
```
模版名称：用药提醒
模版内容：
药方名称{{thing9.DATA}}
服药时间{{time1.DATA}}
服药人{{thing4.DATA}}
提示说明{{thing3.DATA}}
```

### 3.3 提交申请

1. 点击 **选用** 按钮
2. 根据需要删减或调整字段（保留核心字段）
3. 填写模版用途说明：
   ```
   用于提醒家庭成员按时服药，包含药品名称、服药时间、服药人等信息
   ```
4. 点击 **提交** 按钮

### 3.4 获取模版ID

申请成功后，在 **我的模版** 列表中可以看到：
- 模版标题
- 模版ID（格式：`xxxxxxxxxxxxxxxxxxxxxx`）
- 详细内容

**请记录此模版ID，后续代码中需要使用**

---

## 四、申请过期预警模版

### 4.1 搜索模版

在搜索框输入关键词：**到期提醒** 或 **过期提醒**

### 4.2 选择合适的模版

**模版示例：**
```
模版名称：到期提醒
模版内容：
药品名称{{thing1.DATA}}
过期时间{{time2.DATA}}
剩余天数{{number5.DATA}}
温馨提醒{{thing3.DATA}}

```

### 4.3 提交申请

填写模版用途说明：
```
用于提醒用户家中药品即将过期，避免误服过期药品
```

### 4.4 获取模版ID

记录此模版ID，用于药品过期预警功能
