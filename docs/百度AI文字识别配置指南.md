# 药效记 - 百度AI文字识别配置指南

## 一、前置准备

### 1.1 注册百度智能云账号

1. 访问 [https://cloud.baidu.com](https://cloud.baidu.com)
2. 点击右上角 **注册** 按钮
3. 使用手机号或邮箱完成注册
4. 完成实名认证（个人或企业）

### 1.2 所需信息

- 百度智能云账号
- 已完成实名认证
- 手机号（用于接收验证码）

---

## 二、创建文字识别应用

### 2.1 进入文字识别控制台

1. 登录百度智能云控制台
2. 在顶部搜索框输入 **文字识别** 或 **OCR**
3. 点击进入 **文字识别 OCR** 产品页
4. 点击 **立即使用** 按钮

### 2.2 开通服务

1. 首次使用需要开通服务
2. 阅读并同意服务协议
3. 点击 **开通** 按钮

### 2.3 创建应用

1. 在左侧菜单选择 **应用列表**
2. 点击 **创建应用** 按钮
3. 填写应用信息：
   - **应用名称**：`药效记-药品识别`
   - **应用类型**：选择 **通用文字识别**
   - **接口选择**：勾选以下接口
     - ✅ 通用文字识别（标准版）
     - ✅ 通用文字识别（高精度版）
   - **应用归属**：个人/企业
   - **应用描述**：`用于识别药品包装上的文字信息`

4. 点击 **立即创建** 按钮

### 2.4 获取API密钥

创建成功后，在应用列表中可以看到：
- **AppID**：应用ID
- **API Key**：API密钥
- **Secret Key**：密钥

**请妥善保管这些信息，后续代码中需要使用**
