# 编译并运行药效记小程序计划

## 项目分析
项目是一个基于微信小程序的家庭常备药品智能管理与过期提醒系统，包含前端和后端两部分。

## 运行步骤

### 1. 安装依赖
- **前端依赖**：在 `miniprogram` 目录下运行 `npm install`
- **后端依赖**：在 `server/HomeMediRemind` 目录下运行 `npm install`

### 2. 配置数据库
- 确保本地 MySQL 服务已启动
- 创建数据库 `family_medicine_db`
- 验证 `server/HomeMediRemind/db.js` 中的数据库配置（用户名、密码等）

### 3. 启动后端服务
- **方法1**：运行 `server/HomeMediRemind/start-server.bat` 脚本
- **方法2**：在 `server/HomeMediRemind` 目录下运行 `node app.js`
- 服务将在 http://localhost:3001 启动

### 4. 运行前端
- 使用微信开发者工具打开 `miniprogram` 目录
- 配置 AppID（如果没有可使用测试号）
- 执行 "构建 npm" 操作，生成 `miniprogram_npm` 目录
- 点击编译按钮运行小程序

## 注意事项
- 确保 MySQL 服务正常运行
- 确保后端服务启动成功
- 确保微信开发者工具已安装并登录
- 首次运行需要安装依赖并构建 npm

## 技术栈
- **前端**：微信小程序原生开发 + Vant Weapp 组件库
- **后端**：Node.js + Express + MySQL
- **通信**：HTTP API + MQTT 实时消息推送
