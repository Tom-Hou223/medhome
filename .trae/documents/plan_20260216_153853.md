# 修复年月选择器可选范围问题

## 问题分析

用户反映年月选择器中只有当前月份（2026年2月）的选项，无法选择其他年份和月份。

### 根本原因

* 为了让日历只显示一个月份，我们修改了 `initDate` 函数，将 `minDate` 和 `maxDate` 设置为只包含当前月份
* 这也影响了年月选择器的可选范围，导致选择器只能显示当前月份

## 解决方案

### 1. 分离日历和选择器的日期范围

* 为年月选择器设置独立的日期范围，从当前年份前5年到后5年
* 保持日历的日期范围只包含当前月份

### 2. 修改代码结构

* 在 `data` 中添加新的字段 `pickerMinDate` 和 `pickerMaxDate`，用于年月选择器
* 修改 `initDate` 函数，同时设置日历和选择器的日期范围
* 修改 `van-datetime-picker` 组件，使用独立的日期范围

### 3. 保持功能兼容性

* 确保修改后的配置不会影响其他功能
* 保持日历只显示一个月份的功能
* 确保年月选择器能够显示多个年份和月份的选项

## 技术实现

### 修改data结构

```javascript
data: {
  // 其他字段...
  minDate: null,       // 日历的最小日期（只包含当前月份）
  maxDate: null,       // 日历的最大日期（只包含当前月份）
  pickerMinDate: null, // 年月选择器的最小日期（前5年）
  pickerMaxDate: null, // 年月选择器的最大日期（后5年）
}
```

### 修改initDate函数

```javascript
initDate: function() {
  const today = new Date();
  const year = today.getFullYear();
  const monthNumber = today.getMonth() + 1;
  const monthStr = String(monthNumber).padStart(2, '0');
  const dayStr = String(today.getDate()).padStart(2, '0');
  const selectedDate = `${year}-${monthStr}-${dayStr}`;
  const baseDate = today.getTime();

  // 日历的日期范围：只包含当前月份
  const monthFirstDate = new Date(year, monthNumber - 1, 1);
  const monthLastDay = new Date(year, monthNumber, 0).getDate();
  const monthLastDate = new Date(year, monthNumber - 1, monthLastDay);
  const minDate = monthFirstDate.getTime();
  const maxDate = monthLastDate.getTime();

  // 年月选择器的日期范围：前5年到后5年
  const pickerMinYear = year - 5;
  const pickerMaxYear = year + 5;
  const pickerMinDate = new Date(pickerMinYear, 0, 1).getTime();
  const pickerMaxDate = new Date(pickerMaxYear, 11, 31).getTime();

  this.setData({
    currentDate: today,
    selectedDate: selectedDate,
    currentYear: year,
    currentMonth: monthNumber,
    calendarBaseDate: baseDate,
    minDate,
    maxDate,
    pickerMinDate,
    pickerMaxDate
  });

  this.loadMonthStatus(year, monthNumber);
}
```

### 修改van-datetime-picker组件

```xml
<van-datetime-picker
  type="year-month"
  value="{{calendarBaseDate}}"
  min-date="{{pickerMinDate}}"
  max-date="{{pickerMaxDate}}"
  bind:confirm="onYearMonthConfirm"
  bind:cancel="closeYearMonthPicker"
/>
```

## 预期效果

修改后，日历将只显示一个月份，而年月选择器将能够显示从当前年份前5年到后5年的所有年份和月份的选项。

* 确保修改后的配置不会影响其他功能
* 保持代码的可读性和可维护性
