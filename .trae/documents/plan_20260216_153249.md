# 修改日历只显示一个月份

## 问题分析

用户反映日历显示了多个月份（如2021年6月和7月），希望只显示一个月份。

### 根本原因

* 当前日历组件的配置可能没有限制只显示一个月份
* `minDate` 和 `maxDate` 的设置可能允许显示多个月份
* 日历组件的属性配置可能需要调整

## 解决方案

### 1. 修改日历组件配置

* 调整 `van-calendar` 组件的属性，限制只显示一个月份
* 可能需要设置 `row-height` 或其他属性来控制显示

### 2. 重新设置日期范围

* 修改 `initDate` 函数，确保 `minDate` 和 `maxDate` 只包含当前月份
* 确保其他函数（如 `onPrevMonth`、`onNextMonth`、`onYearMonthConfirm`）也使用相同的逻辑

### 3. 保持功能兼容性

* 确保修改后的配置不会影响其他功能
* 保持年月选择器的正常工作
* 确保日历的基本功能正常

## 技术实现

### 修改van-calendar组件配置

```xml
<van-calendar
  poppable="{{false}}"
  show="{{true}}"
  type="single"
  show-confirm="{{false}}"
  show-title="{{false}}"
  show-subtitle="{{false}}"
  color="#27AE60"
  min-date="{{minDate}}"
  max-date="{{maxDate}}"
  default-date="{{calendarBaseDate}}"
  formatter="{{dayFormatter}}"
  bind:select="onCalendarSelect"
  bind:month-show="onCalendarMonthShow"
  row-height="{{60}}"
>
```

### 修改initDate函数

```javascript
initDate: function() {
  const today = new Date();
  const year = today.getFullYear();
  const monthNumber = today.getMonth() + 1;
  const monthStr = String(monthNumber).padStart(2, '0');
  const dayStr = String(today.getDate()).padStart(2, '0');
  const selectedDate = `${year}-${monthStr}-${dayStr}`;
  const baseDate = today.getTime();

  // 只显示当前一个月份：min/max 为本月首尾
  const monthFirstDate = new Date(year, monthNumber - 1, 1);
  const monthLastDay = new Date(year, monthNumber, 0).getDate();
  const monthLastDate = new Date(year, monthNumber - 1, monthLastDay);
  const minDate = monthFirstDate.getTime();
  const maxDate = monthLastDate.getTime();

  this.setData({
    currentDate: today,
    selectedDate: selectedDate,
    currentYear: year,
    currentMonth: monthNumber,
    calendarBaseDate: baseDate,
    minDate,
    maxDate
  });

  this.loadMonthStatus(year, monthNumber);
}
```

### 修改其他函数

确保 `onPrevMonth`、`onNextMonth`、`onYearMonthConfirm` 函数也设置只包含当前月份的日期范围。

## 预期效果

修改后，日历将只显示一个月份，而不是多个月份，用户界面会更加简洁清晰。

## 注意事项

* 确保修改不会影响年月选择器的正常工作
* 保持日历的基本功能正常
* 确保修改后的界面美观合理
